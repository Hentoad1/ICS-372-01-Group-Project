# Brew & Bite Cafe System

## Running the Application
1. Ensure Java 21+ is installed
2. Run: `gradlew run` or `java -jar build/libs/CafeSystem-1.0.0.jar`

## Default Credentials
- **Barista**: username `barista`, password `barista123`
- **Manager**: username `manager`, password `manager123`

## Features
- Customer ordering with customization
- Barista order fulfillment with status updates
- Manager inventory and menu management
- Real-time updates with Observer pattern
- JSON data persistence
